# 書籍・雑誌小売業 × ホワイト × ミニマル

## アトモスフィア / atmosphere

紙の余白と活字の静けさを Web に移した、読書のためのミニマルな書店。白を主役に据え、
装飾を削ぎ落として本そのものを引き立てる。編集的な階層と間で語る静かな佇まい。

- **トーン**: 落ち着いて知的。声を張らず、書影と見出しで語らせる。
- **キーワード**: `editorial` / `quiet` / `whitespace` / `type-first`
- 情報の階層を余白で作り、視線を書影と見出しへ一点ずつ運ぶ。動きは最小限に留める。

## カラーシステム / color-system

無彩色を基調に、極小の面積でだけ差し色を許す。色数を絞ることで活字と書影を主役にする。

| 役割 | トークン | 値 |
| --- | --- | --- |
| Primary | `--color-primary` | `#1a1a1a` |
| Secondary | `--color-secondary` | `#6b6b6b` |
| Accent | `--color-accent` | `#9a6a3c` |
| Neutral | `--color-neutral` | `#e6e6e6` |
| Background | `--color-bg` | `#ffffff` |
| Foreground | `--color-fg` | `#1a1a1a` |

**コントラスト**: 本文は純黒に近い `fg` を純白 `bg` に置き 18:1 超で AAA。Accent（インク寄りの茶）は
リンクと在庫バッジのみに使い、面積を最小化して白の静けさを保つ。

## タイポグラフィ / typography

- **見出しフォント**: `Source Serif 4`（欧文）/ `Noto Serif JP`（和文）— SIL OFL のオープンフォント。
- **本文フォント**: `Source Serif 4` 本文サイズ / 和文は `Noto Serif JP`。キャプションのみ `Inter`。
- **タイプスケール**: `0.8125rem` / `1rem` / `1.333rem` / `1.777rem` / `2.369rem` / `3.157rem`（1.333 比・強い対比）。
- **ウェイト**: `400`（本文）/ `600`（見出し）。行間 1.8、行長 40–75 文字で読み物の可読性を最優先。

## 余白とレイアウト / spacing-layout

- **余白スケール**: `8px` / `16px` / `24px` / `40px` / `64px` / `96px` / `128px`（大胆な余白）。
- **グリッド**: 単段の中央寄せ本文（幅 680px 前後）＋書影一覧は 2–4 カラムの疎なグリッド。
- **コンテナ最大幅**: `680` / `960` / `1200`（px）。読み物は 680px、一覧は 1200px を上限にする。

## 奥行きとシャドウ / depth-shadow

- **エレベーション**: `flat` / `hairline` の 2 段階のみ。影ではなく罫線と余白で階層を作る。
- **シャドウ**: `none` / `0 1px 0 rgba(0,0,0,.08)` / `0 8px 24px rgba(0,0,0,.06)`（モーダルのみ）。
- **角丸**: `0`（書影・カード）/ `2px`（入力）。角を立て、紙面のような端正さを保つ。

## コンポーネント / components

| コンポーネント | 用法 |
| --- | --- |
| BookCard | 書影＋書名＋著者のみ。影なし、`hairline` の下罫線で区切る。 |
| Heading | 大きな serif 見出しで特集・章を示す。1 画面の主役は 1 つ。 |
| Link | 下線付きインクカラー。ホバーで下線を太く、色は変えない。 |
| Quote | 余白を広く取った引用。編集的トーンの核。 |
| Nav | 極細の上部ナビ。カテゴリはテキストリンクのみ。 |

## ガードレール / guardrails

- **推奨 (dos)**: 余白を惜しまない / 書影と活字を主役にする / 差し色は最小面積に留める。
- **禁止 (donts)**: 影の多用や強い装飾 / 3 種以上の書体混在 / 過密なグリッドや詰め込み。
- **アクセシビリティ**: コントラスト AAA 基調、下線でリンクを明示（色のみに依存しない）、
  フォーカスリング可視、`prefers-reduced-motion` 尊重、本文フォント 16px 以上。

## レスポンシブ / responsive

- **ブレークポイント**: `sm` = 480px / `md` = 768px / `lg` = 1024px / `xl` = 1280px。
- 書影一覧は 2 → 3 → 4 カラムへ段階拡張。読み物は常に中央 1 段で行長を保つ。
  余白は幅に応じ `24px → 64px → 96px` と広げ、白の面積を積極的に確保する。

## エージェントガイド / agent-guide

- **用法**: 本 DESIGN.md 全文をシステムプロンプトへ注入し、以降の UI 生成はこのトークン体系の内側で行う。
- **注入**: 白背景・serif 見出し・広い余白を既定とし、コンポーネントはトークン参照のみで組む。
- **制約**:
  - 影・グラデーション・彩度の高い色を追加しない（差し色は Accent のみ）。
  - オープンライセンス以外のフォント名を出力しない。
  - 特定書店・出版社のブランドやトレードドレスを模倣しない。
  - 余白を削って情報密度を上げる方向の改変を避ける。
