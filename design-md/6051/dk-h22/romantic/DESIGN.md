## アトモスフィア / atmosphere
パステルと曲線で甘さと情感を演出する。 ダークトーンの紫を基調色に、業種「ガソリンスタンド」（JSIC 6051）向けの一貫したトーンを敷く。商いの回遊性と比較のしやすさを、選びやすい陳列と明快な価格提示で促す。

- **トーン**: 親しみやすく軽快な語り口。
- **キーワード**: `romantic` / `sweet` / `dreamy` / `tender`

やわらかな距離感を主役に置き、親しみと安心を先に伝える。装飾は最小限に保ちつつ、
丸みと間で壁の低さを演出し、初見の利用者にも入りやすい導入をつくる。

## 奥行きとシャドウ / depth-shadow
柔らかな多段シャドウと大きめの角丸で親しみと奥行きを出す。要素の重なりをエレベーション段階で

- **エレベーション**: `surface` → `raised` → `overlay` → `modal`
- **シャドウ**: `0 1px 2px rgba(17,17,17,0.06)`; `0 6px 16px rgba(17,17,17,0.10)`; `0 16px 40px rgba(17,17,17,0.14)`
- **角丸**: `8px` · `16px` · `24px` · `9999px`

## コンポーネント / components
物販・カート系の画面を組むための基礎コンポーネント在庫。まず必須部品を揃える。

- **button**: primary / secondary / ghost の 3 バリアント。角丸・間隔トークンを参照。
- **input**: ラベル・ヘルプ・エラーの状態を持ち、フォーカスを常時可視化する。
- **card**: サーフェスとエレベーションで階層を表す汎用コンテナ。
- **navigation**: 主要導線。現在地を明示し、モバイルでは折り畳む。
- **modal**: overlay エレベーションで前面化し、フォーカストラップを備える。
- **badge**: 状態・分類の小片。色は役割トークンに限定する。
- **productCard**: 商品の要約カード。画像・名称・価格を役割トークンで組む。
- **cartSummary**: カート内容と合計の要約。数量変更を反映する。
- **quantityStepper**: 数量の増減入力。下限・上限を明示する。
- **priceTag**: 価格表示。通常価格と割引を役割トークンで区別する。
- **checkoutForm**: 購入手続きフォーム。入力状態とエラーを常時可視化する。

これらを土台に、商品カード・数量入力・カートを役割トークンだけで構成し、状態を明示する。

## ガードレール / guardrails
アクセシビリティの最低ラインを必須要件として明示する。

**推奨 (Do)**
- 役割トークン（色・間隔・角丸）経由で参照する
- コントラスト比 AA 以上を確保する
- 定義済みの余白・タイプスケールに従う

- 商標フォント名・他社ブランド名を使わない
- 着想元サイトのトレードドレスを再現しない

**アクセシビリティ**: WCAG 2.1 AA 準拠: 本文コントラスト 4.5:1 以上、フォーカス可視化、タップ領域 44x44px 以上。

## レスポンシブ / responsive
最小幅から段階的に拡張するモバイルファースト。既定は 1 カラムで、幅の増加に応じて段組を増やす。

| ブレークポイント | 最小幅 |
| --- | --- |
| `sm` | 640px |
| `md` | 768px |
| `lg` | 1024px |
| `xl` | 1280px |

**指針**: モバイルファーストで min-width を積み上げ、各段でコンテナ幅を切り替えて行長 45〜75 文字を保つ。

- メディアクエリは `min-width` を基準に積み上げる（`max-width` の上書きを避ける）。
