## アトモスフィア / atmosphere
往年の配色と書体で懐かしさと個性を出す。 ディープトーンの青緑を基調色に、業種「中央銀行」（JSIC 6211）向けの一貫したトーンを敷く。資産と契約の信頼性を、数値の可読性と厳密な表記で担保する。

- **トーン**: 親しみやすく軽快な語り口。
- **キーワード**: `retro` / `vintage` / `throwback` / `old-school`

手仕事の温もりと素材感を主役に据え、やわらかな曲線と自然な階調で親密さを表す。
不揃いを恐れず、人の手の痕跡を魅力として活かし、生活に寄り添う語り口で一貫させる。

## 余白とレイアウト / spacing-layout
反復するモジュールブロックでリズムを刻む構成。ブロック単位で間隔を揃え、明快な区切りを与える。
8px ベースの間隔スケールでモジュール間の整合を保つ。

- **間隔スケール**: `8px` · `16px` · `32px` · `48px` · `64px` · `96px` · `128px`
- **グリッド**: 16 カラム / ガター 20px / 最大 1200px
- **コンテナ最大幅**: 600px / 840px / 1080px / 1200px

各ブロックは同じ間隔規則で積み、視覚的なテンポを一定に保つ。

## 奥行きとシャドウ / depth-shadow
手触りを感じさせる有機的な奥行き表現。大きめの角丸と柔らかなシャドウで親しみと素材感を出す。

- **エレベーション**: `surface` → `raised` → `overlay` → `modal`
- **シャドウ**: `0 2px 6px rgba(17,17,17,0.08)`; `0 10px 24px rgba(17,17,17,0.12)`; `0 24px 56px rgba(17,17,17,0.16)`
- **角丸**: `12px` · `20px` · `32px` · `9999px`

角丸は要素の役割に応じて選び、過度な丸みで機能を損なわない。

## コンポーネント / components
管理・ダッシュボード系の画面を組むための基礎コンポーネント在庫。まず必須部品を揃える。

- **button**: primary / secondary / ghost の 3 バリアント。角丸・間隔トークンを参照。
- **input**: ラベル・ヘルプ・エラーの状態を持ち、フォーカスを常時可視化する。
- **card**: サーフェスとエレベーションで階層を表す汎用コンテナ。
- **navigation**: 主要導線。現在地を明示し、モバイルでは折り畳む。
- **modal**: overlay エレベーションで前面化し、フォーカストラップを備える。
- **badge**: 状態・分類の小片。色は役割トークンに限定する。
- **dataTable**: 整列・ソート可能な表。密度と可読性を両立する。
- **filterBar**: 絞り込み条件の操作列。適用状態を明示する。
- **statCard**: 主要指標の要約カード。値と推移を役割トークンで示す。
- **chart**: データ可視化。色は役割トークンに限定し凡例を添える。
- **pagination**: 大量データの分割ナビ。現在ページを明示する。

これらを土台に、表・フィルタ・詳細パネルを役割トークン（色・間隔・角丸）だけで構成する。

## レスポンシブ / responsive
最小幅から段階的に拡張するモバイルファースト。既定は 1 カラムで、幅の増加に応じて段組を増やす。

| ブレークポイント | 最小幅 |
| --- | --- |
| `sm` | 640px |
| `md` | 768px |
| `lg` | 1024px |
| `xl` | 1280px |

**指針**: モバイルファーストで min-width を積み上げ、各段でコンテナ幅を切り替えて行長 45〜75 文字を保つ。

- メディアクエリは `min-width` を基準に積み上げる（`max-width` の上書きを避ける）。
