## コンポーネント / components

- **button**: primary / secondary / ghost の 3 バリアント。角丸・間隔トークンを参照。
- **input**: ラベル・ヘルプ・エラーの状態を持ち、フォーカスを常時可視化する。
- **card**: サーフェスとエレベーションで階層を表す汎用コンテナ。
- **navigation**: 主要導線。現在地を明示し、モバイルでは折り畳む。
- **modal**: overlay エレベーションで前面化し、フォーカストラップを備える。
- **badge**: 状態・分類の小片。色は役割トークンに限定する。
- **dataTable**: 整列・ソート可能な表。密度と可読性を両立する。
- **filterBar**: 絞り込み条件の操作列。適用状態を明示する。
- **statCard**: 主要指標の要約カード。値と推移を役割トークンで示す。
- **chart**: データ可視化。色は役割トークンに限定し凡例を添える。
- **pagination**: 大量データの分割ナビ。現在ページを明示する。
