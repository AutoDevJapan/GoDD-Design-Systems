# 経営コンサルタント業 × ライトブルー × 信頼

## アトモスフィア / atmosphere

明晰さと誠実さで「任せて安心」を伝えるコンサルティングの世界観。ライトブルーの澄んだ
知性を基調に、整然としたグリッドと十分な余白で思考の透明性を可視化する。

- **トーン**: 落ち着いて論理的。断定しすぎず、根拠と手順で信頼を積み上げる。
- **キーワード**: `clarity` / `credibility` / `structured` / `calm-confidence`
- 装飾より情報設計を優先し、数字・プロセス・実績を読みやすく構造化する。過度な演出を避け、
  一貫したトークンで「ぶれない専門性」を体現する。

## カラーシステム / color-system

役割ベースのトークンで配色を定義する。色名ではなく役割で参照し、テーマ差し替えに耐える構造にする。

| 役割 | トークン | 値 |
| --- | --- | --- |
| Primary | `--color-primary` | `#2f6fb0` |
| Secondary | `--color-secondary` | `#5b8fc9` |
| Accent | `--color-accent` | `#e8a13a` |
| Neutral | `--color-neutral` | `#5a6472` |
| Background | `--color-bg` | `#f5f8fc` |
| Foreground | `--color-fg` | `#1c2733` |

**コントラスト**: 本文 (`fg` on `bg`) は 12:1 超で WCAG AAA。Primary ボタン上の白文字は 4.7:1 で
AA を満たす。Accent は CTA とハイライトに限定し、面積を絞って信頼基調を崩さない。

## タイポグラフィ / typography

- **見出しフォント**: `Inter`（欧文）/ `Noto Sans JP`（和文）— いずれも SIL OFL のオープンフォント。
- **本文フォント**: `Source Sans 3`（欧文）/ `Noto Sans JP`（和文）。
- **タイプスケール**: `0.875rem` / `1rem` / `1.25rem` / `1.5rem` / `2rem` / `2.5rem`（1.25 比）。
- **ウェイト**: `400`（本文）/ `600`（強調・小見出し）/ `700`（見出し）。字間はやや詰め、行間 1.7 で可読性を確保。

## 余白とレイアウト / spacing-layout

- **余白スケール**: `4px` / `8px` / `16px` / `24px` / `32px` / `48px` / `64px`（8px グリッド基調）。
- **グリッド**: 12 カラム、ガター `24px`。セクションは縦リズムを `8px` 倍数で揃え、静かな秩序を作る。
- **コンテナ最大幅**: `640` / `960` / `1200`（px）。読み物系は 720px 前後で最適行長を保つ。

## 奥行きとシャドウ / depth-shadow

- **エレベーション**: `flat` / `raised` / `overlay` の 3 段階。多用せず階層を明確にする。
- **シャドウ**: `0 1px 2px rgba(28,39,51,.06)` / `0 4px 12px rgba(28,39,51,.10)` / `0 12px 32px rgba(28,39,51,.14)`。
- **角丸**: `4px`（入力）/ `8px`（カード）/ `12px`（モーダル）。硬すぎず、堅実さを保つ控えめな丸み。

## コンポーネント / components

| コンポーネント | 用法 |
| --- | --- |
| Button | Primary は 1 画面 1 主行動。Secondary は輪郭線。Accent は資料請求など高価値 CTA に限定。 |
| Card | 実績・サービスを均質なカードで並置。影は `raised` 止まり。 |
| Stat | 数値 KPI を大きく、単位・注釈を小さく。信頼の裏づけに用いる。 |
| Table | 料金・比較は罫線最小の zebra なし表。数値は右揃え。 |
| Nav | 上部固定の水平ナビ。現在地を Primary 下線で明示。 |

## ガードレール / guardrails

- **推奨 (dos)**: 根拠となる数値・出典を併記する / 行動導線を 1 つに絞る / トークン内で配色する。
- **禁止 (donts)**: 誇張グラデーションや過剰アニメーション / Accent の広面積使用 / 3 種を超える書体混在。
- **アクセシビリティ**: コントラスト AA 以上、フォーカスリング可視、`prefers-reduced-motion` を尊重、
  タップ領域 44px 以上、色のみに依存しない情報伝達。

## レスポンシブ / responsive

- **ブレークポイント**: `sm` = 480px / `md` = 768px / `lg` = 1024px / `xl` = 1280px。
- モバイルは 1 カラム、`md` で 2 カラム、`lg` 以上で 12 グリッドを展開。ナビは `md` 未満でドロワーに畳む。
  余白は画面幅に応じて `16px→24px→32px` と段階的に拡張する。

## エージェントガイド / agent-guide

- **用法**: 本 DESIGN.md 全文をシステムプロンプトへ注入し、以降の UI 生成はこのトークン体系の内側で行う。
- **注入**: カラー/タイポ/余白は CSS 変数として先頭で宣言し、コンポーネントはトークン参照のみで組む。
- **制約**:
  - トークン外の色・影・角丸を新規に発明しない。
  - オープンライセンス以外のフォント名を出力しない。
  - 特定企業のブランド・ロゴ・トレードドレスを模倣しない。
  - 信頼基調を損なう演出（点滅・過度な余白圧縮・低コントラスト）を避ける。
