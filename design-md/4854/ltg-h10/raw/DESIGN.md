## 奥行きとシャドウ / depth-shadow

- **エレベーション**: `surface` → `raised` → `overlay` → `modal`
- **境界**: `1px solid` のヘアラインとサーフェス塗りで階層を表す（影は原則使わない）。
- **シャドウ**: `none`; `0 0 0 1px rgba(17,17,17,0.12)`; `0 0 0 1px rgba(17,17,17,0.20)`
