## アトモスフィア / atmosphere
肩肘張らない配色とリズムで気軽さを出す。 ディープトーンの紫みの赤を基調色に、業種「他に分類されない専門サービス業」（JSIC 7299）向けの一貫したトーンを敷く。専門知の厳密さと出典性を、抑制された論理的構成と静かな余白で示す。

- **トーン**: 親しみやすく軽快な語り口。
- **キーワード**: `casual` / `relaxed` / `easygoing` / `informal`

やわらかな距離感を主役に置き、親しみと安心を先に伝える。装飾は最小限に保ちつつ、
丸みと間で壁の低さを演出し、初見の利用者にも入りやすい導入をつくる。

## 余白とレイアウト / spacing-layout
余白を主役にした流動的なキャンバス構成。要素同士の間隔で階層を示し、開放感を優先する。
8px ベースの間隔スケールで一貫したリズムを敷く。

- **間隔スケール**: `12px` · `20px` · `32px` · `48px` · `72px` · `112px` · `160px`
- **グリッド**: 流動グリッド / ガター 24px / 最大 1440px
- **コンテナ最大幅**: 720px / 960px / 1200px / 1440px

コンテナ幅は広めに取り、要素を余白に浮かべるように配置する。

## 奥行きとシャドウ / depth-shadow
柔らかな多段シャドウと大きめの角丸で親しみと奥行きを出す。要素の重なりをエレベーション段階で
表し、浮遊感を程よく与える。

- **エレベーション**: `surface` → `raised` → `overlay` → `modal`
- **シャドウ**: `0 1px 2px rgba(17,17,17,0.06)`; `0 6px 16px rgba(17,17,17,0.10)`; `0 16px 40px rgba(17,17,17,0.14)`
- **角丸**: `8px` · `16px` · `24px` · `9999px`

## コンポーネント / components
管理・ダッシュボード系の画面を組むための基礎コンポーネント在庫。まず必須部品を揃える。

- **button**: primary / secondary / ghost の 3 バリアント。角丸・間隔トークンを参照。
- **input**: ラベル・ヘルプ・エラーの状態を持ち、フォーカスを常時可視化する。
- **card**: サーフェスとエレベーションで階層を表す汎用コンテナ。
- **navigation**: 主要導線。現在地を明示し、モバイルでは折り畳む。
- **modal**: overlay エレベーションで前面化し、フォーカストラップを備える。
- **badge**: 状態・分類の小片。色は役割トークンに限定する。
- **dataTable**: 整列・ソート可能な表。密度と可読性を両立する。
- **filterBar**: 絞り込み条件の操作列。適用状態を明示する。
- **statCard**: 主要指標の要約カード。値と推移を役割トークンで示す。
- **chart**: データ可視化。色は役割トークンに限定し凡例を添える。
- **pagination**: 大量データの分割ナビ。現在ページを明示する。

これらを土台に、表・フィルタ・詳細パネルを役割トークン（色・間隔・角丸）だけで構成する。

## ガードレール / guardrails
アクセシビリティの最低ラインを必須要件として明示する。

**推奨 (Do)**
- 役割トークン（色・間隔・角丸）経由で参照する
- コントラスト比 AA 以上を確保する
- 定義済みの余白・タイプスケールに従う

**禁止 (Don't)**
- トークン外のハードコード値を新規に導入しない
- 商標フォント名・他社ブランド名を使わない
- 着想元サイトのトレードドレスを再現しない

**アクセシビリティ**: WCAG 2.1 AA 準拠: 本文コントラスト 4.5:1 以上、フォーカス可視化、タップ領域 44x44px 以上。

- テキストのコントラストは WCAG AA（本文 4.5:1 / 大見出し 3:1）以上。
- フォーカスは常に可視化し、タップ領域は 44x44px 以上を確保する。

## レスポンシブ / responsive
最小幅から段階的に拡張するモバイルファースト。既定は 1 カラムで、幅の増加に応じて段組を増やす。

| ブレークポイント | 最小幅 |
| --- | --- |
| `sm` | 640px |
| `md` | 768px |
| `lg` | 1024px |
| `xl` | 1280px |

**指針**: モバイルファーストで min-width を積み上げ、各段でコンテナ幅を切り替えて行長 45〜75 文字を保つ。

- メディアクエリは `min-width` を基準に積み上げる（`max-width` の上書きを避ける）。
